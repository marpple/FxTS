Lazy evaluation

Lazy evaluation is possible, It will consume "Iterable/asyncIterable" when it needs to be evaluated. so efficient calculation is attainable in declaratively written code.

pipe(
  [1, 2, 3, 4, 5],
  map(a => a + 10),
  filter(a => a % 2 === 0),
  take(2),
  toArray
); // [12, 14]
