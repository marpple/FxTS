# FxTS website

## Prerequisite

- api-extractor
  ```
  npm install -g @microsoft/api-extractor@7.52.6
  ```
- api-documenter
  ```
  npm install -g @microsoft/api-documenter@7.26.26
  ```

## Install / Update plugin markdown-plugin

```
$ make install
$ make plugin
```

## Document build

```
$ make build
```

## Start

```
$ npm run start
```
